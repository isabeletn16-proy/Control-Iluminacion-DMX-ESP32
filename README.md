# Proyecto: Sistema de Control de Iluminación Escénica

Diseño y desarrollo de un sistema de control DMX basado en ESP32 para luminarias Chauvet DJ Wash FX.

## Estructura del proyecto
- `/firmware`: Código fuente para el microcontrolador ESP32 (Arduino IDE).
- `/interfaz`: Aplicación en Python para el control de iluminación.

## Tecnologías utilizadas
- Microcontrolador: ESP32-WROOM-32
- Lenguajes: C++ (Arduino), Python (Tkinter)
- Protocolo: RS-485 / DMX512
